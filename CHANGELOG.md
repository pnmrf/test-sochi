# ВМАС — История изменений

## 2026-04-21

### Changed
- **app.js** — расширен с ~1550 до 1945 строк (добавлена логика для геолокации, 3D-режима, нейрохроник, preview-маркеров)
  - **CONFIG** — добавлены параметры `bearing`, `pitch`, `previewZoomOut`
  - **STATE** — существенно расширено новыми полями для управления геолокацией, 3D-просмотром, режимами нейрохроник
- **style.css** — расширен с ~2200 до 2856 строк (новые стили для легенды, preview-режима маркеров, улучшения интерфейса)
- **Структура данных** — изменены связи между объектами и нейрохониками:
  - `objects.json`: добавлено поле `chronicle_ids` для связи с нейрохониками
  - `neurochronicles.json`: переработана структура — добавлены поля `title`, `short_desc`, `lat`, `lng`, `year`, `status`, `object_ids`, `wiki_slug`
- **Функции-хелперы** — добавлены новые утилиты:
  - `getAuthorIds()` — получение ID авторов с поддержкой обоих форматов (`author_ids` и `architect_id`)
  - `hexToRgba()` — преобразование цветов для динамических стилей
  - `getPrimaryImage()` — единая логика выбора главного изображения
  - `getItemYear()` — поддержка обоих формата года (`year_built` и `year`)
  - `getItemMeta()` — составление метаданных для отображения
  - `updateTopBarTitle()` — автоматизация заголовка top bar в зависимости от вкладки

### Added
- **Геолокация** — кнопка на карте для определения текущей позиции пользователя
- **Легенда на карте** — интерактивная панель с фильтрами по типам объектов
- **3D-режим карты** — поддержка наклона и поворота (pitch, bearing)
- **Preview-режим маркеров** — при приближении к карте маркеры переключаются с точек на preview-карточки
- **Автоматическое открытие легенды** — на десктопе легенда открывается по умолчанию
- **Связь объектов с нейрохониками** — двусторонняя связь через `chronicle_ids` / `object_ids`

### Optimized
- **Хелперы типов** — унификация работы с разными форматами данных (поддержка обратной совместимости)
- **Маркеры на карте** — добавлена логика preview-режима для улучшения UX при приближении

## 2026-04-14

### Added
- **Новая вкладка «Главная»** — первая кнопка в таббаре (иконка `home`)
  - Экран `screen-home` с информацией о проекте
  - Карточка "О проекте" с описанием миссии
  - Блок статистики: 4 карточки (объекты на карте, 3D-модели, нейрохроники, архитекторы)
  - Ссылка на Telegram-канал
  - Секция "Как пользоваться" с 3 пунктами (Карта, Вики, 3D)
  - Подвал с кредитом проекта

### Changed
- **Tab bar** — увеличена ширина индикатора для 4 вкладок: `width: calc((100% - 16px) / 4)`
- **Tab bar max-width** — увеличен на десктопе с 320px на 420px
- **Кнопка в top bar** — логотип музея теперь ведёт на вкладку "Главная" вместо полёта на карту
- **Функция `switchTab()`** — добавлена ветка для `tab === 'home'`
- **Функция `updateTopBarTitle()`** — добавлена ветка для home-вкладки
- **CSS секция 16** — добавлена полная стилизация home-экрана (glassmorphism, responsive grid, animations)

### New Functions
- **`updateHomeStats()`** — динамически обновляет счётчики статистики из `STATE.objects`, `STATE.architects`, `STATE.neurochronicles`
  - Вызывается при загрузке данных и при переходе на home-вкладку
