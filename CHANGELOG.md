# ВМАС — История изменений

## 2026-04-14

### Added
- **Новая вкладка «Главная»** — первая кнопка в таббаре (иконка `home`)
  - Экран `screen-home` с информацией о проекте
  - Карточка "О проекте" с описанием миссии
  - Блок статистики: 4 карточки (объекты на карте, 3D-модели, нейрохроники, архитекторы)
  - Ссылка на Telegram-канал
  - Секция "Как пользоваться" с 3 пунктами (Карта, Вики, 3D)
  - Подвал с кредитом проекта

### Changed
- **Tab bar** — увеличена ширина индикатора для 4 вкладок: `width: calc((100% - 16px) / 4)`
- **Tab bar max-width** — увеличен на десктопе с 320px на 420px
- **Кнопка в top bar** — логотип музея теперь ведёт на вкладку "Главная" вместо полёта на карту
- **Функция `switchTab()`** — добавлена ветка для `tab === 'home'`
- **Функция `updateTopBarTitle()`** — добавлена ветка для home-вкладки
- **CSS секция 16** — добавлена полная стилизация home-экрана (glassmorphism, responsive grid, animations)

### New Functions
- **`updateHomeStats()`** — динамически обновляет счётчики статистики из `STATE.objects`, `STATE.architects`, `STATE.neurochronicles`
  - Вызывается при загрузке данных и при переходе на home-вкладку

### Documentation

- Создан `ROADMAP.md` — дорожная карта с сделанным, в работе и планируемым функционалом
